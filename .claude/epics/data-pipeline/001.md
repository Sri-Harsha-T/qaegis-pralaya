# Task: Synthetic cascade scenario generator

## Description

Create `data/generate_scenario.py` that generates 600 synthetic cascade scenarios using graph-based Monte Carlo simulation and Gaussian copula sampling for feature correlation. Each scenario represents a multi-domain crisis with realistic inter-departmental failure chains.

## Acceptance Criteria
- [ ] Script creates `data/synthetic/flood_grid.json` (200 samples)
- [ ] Script creates `data/synthetic/heat_hospital.json` (200 samples)  
- [ ] Script creates `data/synthetic/cyclone_comms.json` (200 samples)
- [ ] Each sample has exactly 10 features: temperature_c, wind_kmh, precipitation_mm, grid_load_pct, grid_outage_nodes, water_pressure_bar, water_treatment_pct, hospital_bed_pct, ambulance_available, telecom_uptime_pct
- [ ] Each sample has cascade metadata: cascade_type, cascade_risk (float 0-1), propagation_chain (list)
- [ ] Script supports `python data/generate_scenario.py --all` command
- [ ] Features preserve realistic correlations via Gaussian copula

## Technical Details

- Use graph-based Monte Carlo simulation (ADR-0003) for cascade_risk labels
- Implement Gaussian copula via `scipy.stats.norm` + Cholesky decomposition
- Load correlation matrix from `docs/phases/phase-1-data/feature_correlations.json` (create if missing)
- BFS propagation for cascade chain analysis per ADR-0007
- Template-based generation with Gaussian noise for physical plausibility
- Output JSON format with proper typing
