# Task: Cascade label generator (graph simulation)

## Description

Create cascade label generation functionality within `data/generate_scenario.py` that uses graph-based simulation to compute realistic cascade_risk scores and propagation_chain sequences. This depends on the synthetic data generator framework being implemented first.

## Acceptance Criteria
- [ ] NetworkX dependency graph implemented with 5 domains: climate, grid, water, medical, telecom
- [ ] BFS propagation algorithm computes cascade_risk (float 0-1) per ADR-0007
- [ ] Each sample includes propagation_chain (ordered list of affected domains)
- [ ] Risk calculation caps at 1.0: `propagated[node] = min(1.0, ...)`
- [ ] Graph edge weights match specification (locked per ADR-0007)
- [ ] Monte Carlo simulation runs 1000+ iterations for label stability

## Technical Details

- Use NetworkX directed graph for dependency modeling
- Edge weights represent failure propagation probability between domains
- BFS traversal from initial failure point (climate trigger)
- Risk accumulation via weighted sum with normalization
- Propagation chain records sequence of domain failures above threshold
- Integration with existing copula sampling from task 001
- Graph structure: climate → grid/water → medical/telecom

- Requires the base scenario generation framework and file structure
