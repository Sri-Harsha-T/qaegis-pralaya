# Task: PennyLane VQC circuit + AngleEmbedding

## Description
Implement the core Variational Quantum Circuit (VQC) using PennyLane with AngleEmbedding for 5-qubit quantum machine learning. This forms the foundation of the QML engine's prediction capabilities.

## Acceptance Criteria
- [ ] VQC circuit implemented with `pennylane.device("default.qubit", wires=5)`
- [ ] AngleEmbedding with X rotation for 5 features
- [ ] StronglyEntanglingLayers ansatz with 2 layers
- [ ] Measurement of 5 PauliZ expectation values
- [ ] Risk mapping from [-1,1] → [0,1] using `(measurement + 1) / 2`
- [ ] PCA preprocessing to reduce 10 features → 5 angles
- [ ] Angle clipping to [0, π] before embedding
- [ ] CascadeRiskScorer class with predict_domain_risks method

## Technical Details
- **Device**: `pennylane.device("default.qubit", wires=5)` (ADR-0001)
- **Encoding**: `qml.AngleEmbedding(features, wires=range(5), rotation="X")`
- **Ansatz**: `qml.StronglyEntanglingLayers(weights, wires=range(5))`, n_layers=2
- **Measurement**: `[qml.expval(qml.PauliZ(i)) for i in range(5)]`
- **Preprocessing**: sklearn PCA to reduce 10 features to 5 angles
- **Output**: Dict with keys: climate, power_grid, water, medical, telecom (all float 0-1)
